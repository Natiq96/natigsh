Html nədir? Kodlaşdırma nədir?

Html nədir?

HTML (HyperText Markup Language) veb səhifələrin strukturunu yaratmaq üçün istifadə olunan əsas işarələmə dilidir. Saytda gördüyümüz başlıqlar, mətnlər, şəkillər və linklər HTML ilə yerləşdirilir. O, proqramlaşdırma dili deyil, strukturlaşdırma dilidir. Veb brauzerlər HTML kodunu oxuyaraq səhifəni vizual şəkildə göstərirlər.

Kodlaşdırma nədir?

Kodlaşdırma kompüterlərlə ünsiyyət qurmaq üçün istifadə olunan təlimatların yazılması prosesidir. Müxtəlif proqramlaşdırma dilləri ilə yazılan bu təlimatlar kompüterlərə nə edəcəyini izah edir. Kodlaşdırma proqramların, saytların və tətbiqlərin işləməsini təmin edir. Hər sahədə avtomatlaşdırma və texnologiyanın əsasını kod təşkil edir.

Css nədir? Project(layihə) nədir?

Css nədir?

CSS (Cascading Style Sheets) veb səhifələrin vizual görünüşünü tənzimləmək üçün istifadə olunan stil dilidir. HTML ilə yaradılan strukturun rəngini, ölçüsünü, şriftini və yerləşimini CSS ilə dəyişmək mümkündür. Bu sayədə saytlar daha cəlbedici və istifadəçi dostu görünür. CSS kodları HTML faylına daxili, xarici və ya inline formada əlavə oluna bilər.

Project(layihə) nədir?

Project (layihə) müəyyən bir məqsədə çatmaq üçün planlanmış və yerinə yetirilən işlər toplusudur. Proqramlaşdırmada project adətən bir tətbiq, veb sayt, oyun və ya proqram yazmaq üçün görülən işləri əhatə edir. Layihələr real bacarıqları inkişaf etdirmək və öyrəndiklərini tətbiq etmək üçün əla üsuldur. Kiçik project-lərlə başlamaq öyrənmə prosesini daha səmərəli edir.

Github nədir?

Github

Git proqramlaşdırmada layihələrin versiya nəzarətini aparmaq üçün istifadə olunan güclü bir sistemdir. O, kodda edilən bütün dəyişiklikləri yadda saxlayır və əvvəlki versiyalara qayıtmaq imkanı verir. Bu, xüsusilə proqramda səhv olduqda və ya dəyişiklikləri müqayisə etmək istədikdə çox faydalıdır. Git həmçinin komandada işləyən proqramçılar üçün böyük rahatlıq yaradır — hər kəs öz kodunu əlavə edə, dəyişiklikləri birləşdirə və konfliktləri həll edə bilər. Layihənin inkişaf mərhələləri asanlıqla izlənilir və sənədləşir. GitHub kimi platformalar vasitəsilə Git istifadə edərək kod paylaşmaq və açıq mənbəli layihələrə töhfə vermək də mümkündür. Bu, həm fərdi, həm də komanda şəklində işləyənlər üçün ideal bir vasitədir.
